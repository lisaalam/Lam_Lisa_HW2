MUSIC PLAYLIST ICONS

HOME ICON

This home icon brings you back to the home page.

SEARCH ICON

This is the search icon that allows you to search songs and playlists.

FAVOURITES ICON

The favourites icon lets you save songs into a favourites playlist.

BACKWARDS ICON

This backwards icon brings you back to the previous song.

PAUSE ICON

This is the pause icon that pauses the current song playing.

PLAY ICON

The play icon lets you play the current song.

FORWARDS ICON

This is the forwards icon brings you to the next song.

ADD ICON

This add icon allows you to add songs to your playlist.

DELETE ICON

The delete icon will delete any song you have on your current playlist.

SAVE ICON

This is the save icon that saves any songs.
